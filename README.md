# 🚀 Learning Platform (MERN Stack)

## 📌 Overview
A full-stack **Learning Management Platform** built using the MERN stack.  
It allows users to explore courses, authenticate securely, and manage learning content through an intuitive UI.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS / CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Authentication
- Clerk (for user authentication & management)

### Deployment
- Frontend: Vercel
- Backend: (Render / Vercel / any you used)
- Database: MongoDB Atlas

---

## ✨ Features

- 🔐 User Authentication (Signup/Login using Clerk)
- 📚 Course Listing & Management
- 🎯 Responsive UI for all devices
- 🔄 API Integration (Frontend ↔ Backend)
- ⚡ Fast and optimized performance
- 🌐 Deployed and accessible online
